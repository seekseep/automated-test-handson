# 03-test-basics

自動テストの基本セクション。`node:test` の最小例・AAA パターン・実行方法を学ぶ。

| レクチャー       | 内容                                                                            |
| ---------------- | ------------------------------------------------------------------------------- |
| `01-first-test`  | はじめてのテスト。`node:test` 最小例を書いて `node --test` / `--watch` で動かす |
| `02-aaa-pattern` | AAA（準備・実行・検証）でテストを読みやすく書く                                 |
| `03-assertions`  | よく使うアサーション（`equal` / `deepEqual` / `ok` / `throws`）の使い分け       |
| `04-good-tests`  | 良いテストの性質（速い・決定的・1つを検証・名前が仕様）                         |
| `05-test-schools` | 古典学派とロンドン学派。同じ題材を「状態の検証」と「相互作用の検証」で書き分ける |
| `06-test-doubles` | テストダブル（モック・スパイ・スタブ・フェイク）を依存差し替えで使い分ける       |

解説: [docs/03_basic.md](../../docs/03_basic.md)
